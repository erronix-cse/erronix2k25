BugBuster

About the Event

BugBuster is a challenging debugging competition that tests programming and problem-solving skills. Participants will be given code snippets containing errors which they must identify and correct, demonstrating logical thinking, coding precision, and efficiency.

  • Participation: Individual only

Registration & Submission

  • Registration Process: Online / First-come, first-served.
  • Registration Deadline: 29/10/2025.
  • Prerequisite: No prior submission required.

General Rules

  1. Only individual participation is allowed.
  2. Participants must use original effort; malpractice leads to disqualification.
  3. The decision of the judges/organizers will be final.

Specific Rules

  1. Participants will be given code snippets containing:
    • Syntax errors
    • Logical errors
  2. Languages allowed: C, C++, Java, Python, or JavaScript.
  3. The same debugging questions will be given for all languages.
  4. Internet usage is strictly prohibited.
  5. Only organizers’ systems may be used.
  6. Language selection must be confirmed before the event begins.

Event Flow & Time Allocation

  • Total Duration: 30–45 minutes (decided by organizers/judges).
  • Format: Single round.

Evaluation Criteria

  • Number of bugs correctly fixed.
  • Time taken to fix the bugs.
  • Ranking is based on fixing more bugs in less time.

Code of Conduct

  • Participants must report on time.
  • Maintain professionalism and decorum throughout the event.
  • Any violation of rules may lead to disqualification.
